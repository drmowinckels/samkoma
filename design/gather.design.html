Modern when2meet alternative
gather

A polished, API-first alternative to when2meet — paint your availability, watch the group heatmap converge, and spin up polls straight from the CLI or a chat / CI command. Hi-fi explorations below; light and dark, desktop and mobile.

00 — Naming the product
Recommended
gather
"find a time, together."

Says exactly what the tool does — it gathers people and times into one place. Plain, friendly, nothing to explain, and a clean verb for the command line: gather new. Stands on its own as a product name.

Alternates
align
line up everyone's calendars

Crisp and verb-y; reads a little corporate.

when
the direct heir to when2meet

Instantly legible; very generic to own.

slate
put it on the slate

Warm and calendar-ish; also means "schedule".

rally
rally everyone to a time

Energetic; leans event & community.

All screens below use gather. Swap the wordmark and the rest holds.

05 — API & CLI · bot integration (Jinx-ready)
API reference
Reference

One poll, two front doors

Everything the web UI does is a thin client over this REST API. Authenticate with a bot token; create, read, and resolve polls.

MethodEndpointDoes
POST/v1/pollsCreate a poll
GET/v1/polls/:idFetch poll + grid state
POST/v1/polls/:id/slotsSubmit availability
GET/v1/polls/:id/bestTop slots, ranked
Create a poll
$ curl -X POST https://gather.so/v1/polls \
-H "Authorization: Bearer $BOT_TOKEN" \
-d '{
"title": "Team offsite",
"days": ["2025-07-15","2025-07-16","2025-07-17"],
"from": "09:00", "to": "15:00", "slot": 30,
"tz": "Europe/Oslo"
}'
{ "id": "9fK2qd", "url": "https://gather.so/e/9fK2qd" }
